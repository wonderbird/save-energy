# Tech Context

## Technologies

-   **.NET 8**: The core application is built using the .NET SDK.
-   **xUnit**: Used for unit testing.
-   **Reqnroll**: Used for Behavior-Driven Development (BDD) and integration tests.
-   **GitHub Actions**: The CI/CD pipeline is managed through GitHub Actions for building, testing, and publishing reports.

## Tools and Practices

-   **CSharpier**: For consistent C# code formatting.
-   **SonarLint**: For static code analysis.
-   **Coverlet**: For test coverage data collection.
-   **ReportGenerator**: For creating human-readable coverage reports.
-   **Code Climate**: For automated code quality and test coverage reporting.
